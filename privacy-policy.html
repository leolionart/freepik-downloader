Freepik Downloader Privacy Policy

Last updated: April 5, 2026

Overview

Freepik Downloader consists of a companion web app and a Chrome extension. The Chrome extension is designed to open your configured downloader app for the currently active Freepik page.

Data the Extension Handles

The Chrome extension handles only the minimum data needed for its single purpose:

  • The current active Freepik tab URL
  • The downloader app URL configured by the user

What We Store

The extension stores the downloader app URL in Chrome storage so it can open the correct web app in future sessions.

What We Do Not Collect

  • We do not collect your Freepik API key in the Chrome extension
  • We do not sell personal data
  • We do not use the data for advertising
  • We do not collect your browsing history beyond the active Freepik tab needed for the feature

How the Web App Works

The companion web app may store your Freepik API key in a cookie on the app's own domain so Cloudflare Pages Functions can validate and download Freepik resources on your behalf. That key is not stored by the extension itself.

Contact

For support or privacy questions, use the support or repository link associated with the published app.
